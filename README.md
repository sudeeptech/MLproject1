# Health Prediction App

This is a Streamlit web app for predicting Diabetes, Heart Disease, and Parkinson's disease using pre-trained machine learning models.

## Folder Structure
- `app.py` : Main Streamlit app
- `models/` : Contains the pre-trained `.sav` ML models
- `requirements.txt` : Dependencies
- `README.md` : Project description
